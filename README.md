# Automated Personal Finance Analysis (Python)

## Overview
This project demonstrates how to automate personal finance analysis using Python.
With one execution, the workflow:
- Cleans and prepares transaction data
- Categorizes expenses
- Creates monthly summaries
- Generates visual reports
- Exports results automatically

All data shown is anonymized and for demonstration purposes only.

## Tech Stack
- Python
- pandas
- matplotlib
- Jupyter Notebook

## Key Features
- Rule-based expense categorization
- Monthly expense aggregation
- Visual reporting (PNG exports)
- One-click execution
- Clean, reusable notebook structure
